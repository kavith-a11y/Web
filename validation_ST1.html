Home Page validation report

After validating my Home Page, I found one error due to a space in the logo filename (Blue Future Logo.png). I fixed it by renaming the file to BlueFuture_Logo.png. The page now validates with no errors. This experience taught me the importance of proper file naming and clean HTML structure for accessibility and standards compliance.

Gallery Page validation report

When validating my Gallery Page, I found two errors. The first was the same logo filename issue (spaces in Blue Future Logo.png) which I fixed by renaming the file. The second error was an empty src="" attribute in the modal image tag. I resolved this by adding a transparent SVG placeholder, which passes validation while keeping the modal functional. After these fixes, my Gallery Page validated successfully with no errors.

Content Page validation report

When validating my Content Page, I received a warning about multiple h1 headings because both the logo and page title used h1 tags. I fixed this by changing the page title heading to h2 , ensuring only one h1 exists per page. I also corrected the logo filename issue that affected all pages. After these changes, my Content Page validated successfully. This taught me the importance of proper heading hierarchy for accessibility and screen reader users.
