Gallery Page

Technical Description

The Gallery page was built using HTML5 elements like header, nav, main and footer to keep the layout clear and organised. These elements help to create an easy-to-follow design. The image thumbnails are placed inside a div container. Use CSS Grid to create a responsive layout that works well on different devices. The thumbnails include an image, a heading and a short description. They also use data-index attributes to connect with a JavaScript array that stores information about each image. A hidden modal window is used to show content without reloading the page. The modals visibility is controlled using JavaScript, which changes its display from none to flex. The interactive features on the page such as changing text, images and styles are implemented using JavaScript methods like addEventListener and DOM manipulation. These methods allow the page to update content and apply colours and font styles. The Gallery page combines HTML, responsive CSS and JavaScript interactivity to create a functional and user-friendly experience. The pages design and features work together to make it easy for users to navigate and enjoy the gallery. The use of HTML CSS and JavaScript helps to create an engaging experience, for users.

The headings are organized in a way that shows what is most important and helps make the page look neat.
We also made a menu that lets users jump to parts of the page quickly using links. This makes it easier for people to use the page and have an experience. The Back to button is always visible at the bottom of the screen so users can easily go back to the top of the page.
The style of the page was made using a stylesheet called style.css which makes all the pages look the same and also some special styles just for this page. We used something called Flexbox to make some parts of the page like the image gallery and stats look good, on size screens. This helps with spacing and making things look neat. When you put your mouse over some pictures We get a little bigger which makes the page more fun to use. Overall the combination of HTML and flexible styles makes the page work well and look good.
